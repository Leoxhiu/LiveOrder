# LiveOrder
This is a purchase order (PO) management system developed by using **C++** based on console for smoothing the business process.

IDE: Visual Studio 2022

## Features
#### Manager
1. Login
2. Order Management
3. Executive Registration
4. Report Generation

#### Executive
1. Login
2. Order Management
3. Report Generation


## Data structures:
1. Linked List
2. Dynamic Array

## Algorithms:

3. Merge sort

#### Search:
1. Linear search
2. Binary search
